Namespace: jetface(http://jetface.org/JetFace1.0/)

This namespace contains core jetface elements

Actions
actionallows to contribute action to current action contribution context. Action "class" attribute must have class, implementing com.onpositive.semantic.model.ui.property.editors.IBindedActionDelegate
add-fromRealmallows to add action which allows addition of elements from realm to the parent viewer
add-newallows to add new object to the parent viewer
binded-actionallows to contribute element
close-shellallows to close shell were this action is located
commitallows to commit data in the parent binding
composite-actionallows to execute sequence of actions
contributionElementabstract super element of all contribution elements
create-memberallows to add new data object to parent viewer
delete-selectedallows to delete selected data objects
deselectAllallows to deselect all elements in the parent viewer
doNotGroupallows to remove grouping from the parent viewer
edit-selectionallows to start editing of the selected element
groupByallows to create group by action in the parent action context (should be used only inside list or table)
moveDownallows to add moveDown action to the parent viewer
moveUpallows to add move up action to the parent viewer
objectContributionallows to contribute selection specific actions to the current action context
remove-selectionallows to remove selected elements from the parent viewer
selectAllthis action allows to select all elements in the parent viewer
toggle-filterallows to specify boolean filter action, parent control must implement IFiltrable interface
undoredo-blockThis block allows contribute 'undo' and 'redo' actions to the current context


Model
abstractBindingabstract super element of all binding elements
actionsallows to specify a group of actions that should be contributed to parent action context
bindingBinding element allows to create point for data binding
bindingMemberabstract super elements for all elements that may be placed in the binding
content-assistallows to specify content assist for the binding
defaultValueallows to specify default value for the parent binding
external-realmallows to specify external realm for the parent binding
factory-providerallows to connect binding to factory provider which knows how to produce new objects
label-lookupallows to connect binding to mapper that will map text to object of required class
modelallows to describe used data model
objectValueallows to specify binding with a given value
rangeAllows to specify numeric range for a value
realmallows to specify a realm for the given binding
realmDefinitionallows to specify realm which contains string members for the given binding
regexp-validatorallows to add validator based on validation of text representation of the data against given regular expression
separator-charactersallows to specify separator characters for the parent binding
stringMemberAllows to add string value to the realm
text-labelallows to setup custom text label provider on a element
uniqueMarks that this element property should have unique value
validator'validator' element allows to install custom value validator on the binding


Settings
columnspecifies column in the table
columnscolumn that are represented in the parent table should be described inside this element
command-handlerallows to declare handler for a command (eclipse commands)
contentProviderallows to specify custom content provider for the parent viewer
data-sourcewill be described later
decoratorallows to add decorator to the parent viewer
interceptorallows to add action interceptor to the parent viewer
labelProviderallows to specify custom label provider for the parent viewer
popup-menuAllows to specify popup menu for element
settingabstract super element for settings
toolbar-managerit is needed to investigate what it is for
tooltip-creatorallows to setup custom tooltip creator


UI Controls
browserAllows to insert browser control to the parent container
buttonDescription is not specified
checkboxAllows user to add check box editor to the parent container
colorSelectorUsed for creating special color select button
comboallows to add combo editor to the parent container
composite-editordeclares complex editor
containerbasic container for UI controls
ctab-folderallows to create CTabFolder control
dateallows to add date time control to the parent container
editorabstract super elements of all elements that may be binded to some binding in the data model
formallows to create form
groupThis element allows to create SWT Group
hc'hc' element allows to define container with children that will be situated on one line'hc' element allows to define container with children that will be situated on a one per line basis
headControlallows to specify container that should be placed inside head control of parent form
hgroupThis element allows to create horizontally oriented SWT Group
labelallows to insert text label to the parent control
linkallows to add link control to the parent container
listallows to insert a list control to the parent container
passwordallows to add password editor to the parent container
pattern-filterallows to add simple text filter to parent container
propertiesthis control will be described later
radioAllows user to create radio button
scrollableAllows to create scrollable container
sectionthis complex editor allows to create section container
separatorAllows to add separator element to the parent container
spinnerallows to add spinner editor to parent container
splitterSplitter allows to create hozizontally or vertically oriented split pane
stackStack container shows first enabled child, it listens to element enablement and is able to adapt to enablement changes
stringsimple text field, supports validation and content assist
tab-folderAllows user to create tab folder
tabletable element allows to insert table control in the parent container
textallows to create control which may be used to represent text and supports configuration using org.eclipse.jface.text.source.SourceViewerConfiguration objects
toolbarthis element allows to insert toolbar control in the current container
treeTree element allows to show a tree to the user
uielementabstract super element for all UI controls
vc'vc' element allows to define container with children that will be situated on a one per line basis
viewerviewer element is abstract super element of all viewers.
