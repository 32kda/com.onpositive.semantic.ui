Abstract Element: uielement

Super elements: None Allowed childs: setting

Direct known sub elements: browser container editor link pattern-filter separator toolbar

abstract super element for all UI controls

Layout attributes summary
visibilityControls conditions when element is visible (not visible elements are removed from layout and did not consume space)


Uncategorized attributes summary
backgroundbackground attribute allows to specify background color from JFace color registry that will be used for the element
background-imagebackground image allows to specify background image that will be used for the element value should be identifier of the image declared using 'com.onpositive.semantic.ui.images' extension point
captionlabel for the component(may be interpreted differently depending of component implementation)
enablementallows to specify expression that controls the should be given element enabled or not
fontfont allows to specify font that will be used for the element(expect key that may be mapped to font in JFace font registry)
foregroundforeground allows to specify foreground color from JFace color registry that will be used for the element
grabHorizontalif set to true component will grab horizontal space if it is placed in container with automatic layout management
grabVerticalif set to true component will grab vertical space if it is placed in container with automatic layout management
hAlignHorizontal alignment of the element (is taken in account by auto layouts)
hintDesired size for the component(width,height)
idid attribute allows to refer on it from external context or from same 'dlf' file
indentallows to setup indentation(x indentation, y indentation) of a component in the container will take in container with automatic layout
minimumSizeMinimum size for the component(width,height)
rolerole allows to specify role that will be used for the element
spanallows to setup how many cells container will take in container with automatic layout
texttext to show in the component(may be interpreted differently)
themetheme allows to specify theme that will be used for the element
tooltip-texttooltip text that should be shown when user hovers the element
vAlignVertical alignment of the element (is taken in account by auto layouts) valid values are: fill,top,bottom,center
